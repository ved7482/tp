AWS EC2 Deployment Process

Step 1: Searching for the EC2 Service

  1. Navigate to the AWS Management Console and search for "EC2" in the search bar at the top.

Searching for EC2 Service

Step 2: Launch an Instance

  1. Click on the "Launch Instance" button to begin the EC2 instance setup process.

Launch Instance Button

Name and Tags

  1. Enter a descriptive name for your instance (e.g., "Linux Instance")

Configuring Instance Name and Tags

Selecting Amazon Machine Image (AMI)

  1. Search for "Amazon Linux" in the AMI marketplace
  2. Select the official Amazon Linux AMI

Instance Type

  1. Select t2.micro instance type (free tier eligible)

Network Settings

  1. Use default VPC and subnet
  2. Enable "Auto-assign Public IP"
  3. Create a new security group allowing SSH traffic (port 22)

Configure Storage

  1. Accept the default storage (8 GB) or modify as needed

Launch Instance

  1. Review your instance configuration and click "Launch Instance" to create it

Reviewing and Launching Instance

Step 3: Instance Launched Successfully

  1. Verify that your instance is running in the EC2 dashboard

Step 4: Connect to the Instance

  1. Select your instance and click "Connect" to access the connection options

Connecting to Instance

Step 5: Connect Using EC2 Instance Connect

  1. Choose the EC2 Instance Connect option
  2. Click "Connect" to open a terminal in your browser

Using EC2 Instance Connect

Step 6: Linux Terminal Access

  1. You will now see the Linux terminal in your browser
  2. You are connected to your AWS Linux instance

Linux Terminal Access

Step 7: Executing Linux Commands

  1. Execute the following basic Linux commands to test your instance:
